# 🚀 practicaNextJs

Repositorio de prácticas y ejercicios desarrollados con **Next.js** y **TypeScript**, enfocados en fortalecer habilidades en desarrollo frontend moderno.

## 📚 Objetivo

Aplicar conceptos clave como:

* Tipado avanzado con TypeScript
* Manejo de props y estado en React
* Componentes reutilizables
* Lazy loading y optimización de rendimiento
* Buenas prácticas en estructura de proyectos

## 🛠️ Tecnologías

* Next.js
* React
* TypeScript
* CSS / Tailwind (opcional según práctica)

## 📂 Estructura

Cada carpeta representa una práctica o módulo específico del aprendizaje.

```
/practica-1
/practica-2
/practica-3
```

## ⚙️ Instalación

```bash
git clone https://github.com/tu-usuario/practicaNextJs.git
cd practicaNextJs
npm install
npm run dev
```

## 🎯 Enfoque

Este repositorio está orientado a:

* Aprendizaje práctico
* Mejora continua
* Preparación para proyectos reales y entorno laboral

## 📌 Notas

El contenido se irá actualizando progresivamente a medida que se desarrollen nuevas prácticas.
